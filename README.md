# Todo App

A simple and responsive **Todo Application built with React** that allows users to add and delete tasks with a date. This project demonstrates the use of **React components, props, state management, and event handling**.

---

## 🚀 Features

* Add new todo tasks with a date
* Delete existing tasks
* Dynamic UI updates using React state
* Component-based architecture
* Clean and simple user interface

---

## 🛠️ Technologies Used

* **React.js**
* **JavaScript**
* **CSS**
* **Tailwind (for layout styling)**

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Todo-App.git
```

2. Navigate to the project directory

```bash
cd Todo-App
```

3. Install dependencies

```bash
npm install
```

---

## ▶️ Run the Project

Start the development server:

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```
